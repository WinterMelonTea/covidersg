Global COVID-19 Data Graphs

Graph 1: Cases Statistics

The following graph will show totals and new cases of infections, deaths and recoveries, as well as the active cases (total cases - deaths - recovered). Please click on the box with the country names to select a particular country, or select 'World' for a global outlook.
